# My-Diary-Website

Welcome to the Online Diary Application! This project was developed as a mini project during my studies in 2021 to provide users with a secure and user-friendly platform for writing and managing their diary entries online.

## Features

- **User-Friendly Interface:** 
  - Designed using HTML, CSS, and Bootstrap for an intuitive and responsive user experience.
  - Simple and elegant design to facilitate easy navigation and diary entry management.

- **Diary Entry Creation and Management:**
  - Create, edit, and delete diary entries with ease.
  - Access your entries anytime, anywhere.

- **Backend Functionalities:**
  - Developed using JSP for dynamic content generation.
  - Interacts with the database to store and retrieve diary entries securely.

- **Database Management:**
  - Utilized XAMPP Server to manage the MySQL database.
  - Ensured data security and privacy with appropriate authentication and access control measures.

## Technologies Used

### Frontend:
- **HTML5**
- **CSS3**
- **Bootstrap**
- **JavaScript**

### Backend:
- **Java Server Pages (JSP)**
- **JDBC**

### Database:
- **MySQL (Managed using XAMPP Server)**
